This repository contains the frontend of **Task Master**, a real-time task management application built using **React (Vite)**.  
The frontend communicates with the backend using REST APIs and Socket.IO for real-time updates.

Features

- Create, update, and delete tasks
- Filter tasks by status
- Real-time task updates
- Clean and responsive UI
- Environment-based API configuration
- Deployed on Vercel

Tech Stack

- React (Vite)
- JavaScript
- Axios
- Socket.IO Client
- CSS

live site link: https://task-manager-gamma-one-97.vercel.app/
